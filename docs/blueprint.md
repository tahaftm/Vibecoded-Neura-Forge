# **App Name**: NeuraForge Home

## Core Features:

- Hero Section with Typing Effect: Dynamic title section on the homepage that showcases NeuraForge's services with a backspacing and rewriting effect.
- About Us Button: A prominent button linking to the 'About Us' page, providing more information about the company.
- CEO Profile: Dedicated section displaying information about the CEO Taha Tariq, including his picture, a short bio (BS software engineering at comsats university), and links to his LinkedIn, GitHub, and email.
- Automated Image Generation: Generates and displays a series of random neon robots reflecting automation and Machine Learning themes as the user scrolls the page.
- FAQ Section: A collapsable section addressing frequently asked questions.

## Style Guidelines:

- Primary color: A deep charcoal gray (#333333) for a sophisticated dark theme.
- Secondary color: A vibrant purple (#800080) to add a touch of creativity and highlight interactive elements.
- Background color: A slightly lighter gray (#444444) to provide contrast without being too harsh.
- Body and headline font: 'Roboto', a clean and modern sans-serif font, for readability and a tech-forward feel.
- Use minimalist, neon-style icons related to AI, automation, and data, with a purple or teal accent.
- Modern and clean layout with a focus on readability and visual hierarchy, optimized for a dark theme.
- Subtle animations on page load and when hovering over interactive elements, with a focus on smooth transitions and a high-tech feel.
